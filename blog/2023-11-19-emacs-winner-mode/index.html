Emacs winner-mode to restore your window layouts

If you've been using Emacs for a while, I bet you must have encountered the annoying problem that your Emacs window layout gets messed up by some operations, such as looking for a help, or checking things in magit.

It's ok for me if we can get back to a previous layout, at least for me. Emacs has built in a package for this package: winner-mode. It's very simple, it only have three commands:

  1. M-x winner-mode, to enable/disable the function
  2. M-x winner-undo, to go back to previous layout, execute it repeatedly to go back further
  3. M-x winner-redo, to go back again to a recent layout Unfortunately, you can execute it more than once in a row

For winner-undo, it's a little bit tedious to do if you have to go back a long way, but we can improve it with the help of hydra:

(defhydra hydra (global-map "C-c w")
  "Navigate the time machine of the window layout"
  ("p" winner-undo "Previous layout")
  ("n" winner-redo "Next layout"))

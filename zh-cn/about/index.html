关于

嗨，我是 Ken ，目前在国内一家 BAT 公司搞后台开发；业余时间学习折腾 Emacs Clojure Python ，以及前端开发。

另外，我正在制作分享 Emacs 的视频，发布在油管频道、bilibili（B站）中，欢迎订阅、评论交流和分享转发。

贡献过的一些开源项目

  • 🍪Qtile 一个用 Python 开发和配置的平铺窗口管理器（tiling window manager）
  • Doxygen 从带注释的 C++ 源码生成项目文档的（事实标准的）工具

一些项目

  • text toolkit 在线文本工具箱，目前有两个功能：

    1. 为使用 GNU Make 的 C/C++ 项目生成 compile_commands.json
    2. 使用正则表达式从文本提取内容，比如 IP 、Email、手机号码等
    3. 用文本来表示 ER图
  • ppcompile 一个帮你在本机用 Emacs 写代码，在别的机器上编译代码。他可以帮助你快速定位编译错误，杜绝肉眼对行数，防止视觉疲劳，提升效率。
  • babashka-tools 使用 Babashka 实现的 CLI 工具集（比如密码生成器）
  • extensible-tools 不仅仅止于管理 dotfiles ，还能进行定制扩展以适应不同的使用环境
